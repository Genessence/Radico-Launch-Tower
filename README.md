# Radico Launch Tower

Brand Launch Control Tower — Radico Khaitan. A client-side React dashboard to track alcohol brand launches from ideation to market, with role-based views and a premium liquor-inspired design.

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Setup

```sh
npm i
npm run dev
```

## Scripts

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — run ESLint
- `npm run test` — run tests
